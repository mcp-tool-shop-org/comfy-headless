# comfy-headless

Production-ready headless client for ComfyUI with AI-powered prompt intelligence, video generation, and modular architecture.

## Key Features

- **Headless ComfyUI** — Run ComfyUI workflows without the browser UI
- **AI Prompt Intelligence** — Ollama-powered prompt enhancement and routing
- **Video Generation** — First-class support for video workflows
- **Modular Architecture** — Install only what you need (ai, websocket, health, ui, validation, observability)
- **CLI & API** — Use as a command-line tool or Python library

## Install

```bash
pip install comfy-headless
pip install comfy-headless[standard]  # Recommended: includes AI + WebSocket
pip install comfy-headless[full]      # Everything
```
