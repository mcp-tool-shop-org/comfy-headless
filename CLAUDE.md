# comfy-headless

## What This Does (Prototype)

⚠️ **Prototype Status** — Headless ComfyUI integration for programmatic image generation.

Provides programmatic access to ComfyUI image generation without GUI,
enabling automated workflows and batch processing.

## Features

- Headless ComfyUI interface
- Batch image generation
- Workflow automation
- API-first design
- Model management

## Architecture

- ComfyUI wrapper
- API server
- Workflow executor
- Model handler
- Queue management

## Key Notes

- Prototype ComfyUI wrapper
- Python-first API
- Model management
- Requires ComfyUI backend
